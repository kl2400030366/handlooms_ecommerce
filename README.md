# Handloom Global

A web application to display and sell handloom fashion products to buyers around the world. The platform highlights the uniqueness of handloom items and helps artisans reach a global audience.

## Features

- Admin: Manage platform operations, user roles, and content accuracy.
- Artisan: List products, manage inventory, and interact with buyers.
- Buyer: Browse and purchase handloom items, track orders.
- Marketing Specialist: Promote products globally.
